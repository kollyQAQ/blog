####  建立目录

```shell
cd /usr/lib/
mkdir elasticsearch
```

#### 下载软件包

```shell
cd /usr/src

wget  https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-7.2.0-linux-x86_64.tar.gz

tar zxvf elasticsearch-7.2.0-linux-x86_64.tar.gz -C /usr/lib/elasticsearch/
```
